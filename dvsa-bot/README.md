# DVSA Driving Test Reschedule Bot

An automated Python bot that monitors and reschedules driving tests on the DVSA website.

## Features

- Automatic test availability checking
- Smart calendar scanning with date preferences
- Automated booking with captcha solving
- SMS notifications via Twilio
- Firewall and queue handling
- Multiple test centre support
- Comprehensive logging and error tracking
- Modular architecture for easy maintenance
